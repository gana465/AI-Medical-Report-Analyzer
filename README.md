🩺 Medical Report Analyzer AI
📌 Overview

Medical Report Analyzer AI is a healthcare-focused web application built using Python and Streamlit that analyzes patient medical parameters to assess the risk of kidney-related diseases. The system provides an interactive dashboard where users can enter medical test values and receive an instant risk assessment along with a detailed health summary.

🚀 Features
Interactive and user-friendly dashboard
Patient medical data analysis
Kidney disease risk assessment
Risk score generation
Low, Moderate, and High risk classification
Real-time health insights
Machine Learning integration using Scikit-learn
Automated data preprocessing pipeline
🏥 Diseases Supported
