# 🩺 Medical Report Analyzer AI

## Overview
Medical Report Analyzer AI is a healthcare application built using Python and Streamlit that analyzes patient medical parameters and assesses the risk of kidney-related diseases. The system provides instant health insights through an interactive dashboard.

## Features
- Interactive Streamlit dashboard
- Patient health data analysis
- Kidney disease risk assessment
- Risk score generation
- Low, Moderate, and High risk classification
- Real-time health insights
- Machine learning integration

## Diseases Supported
- Chronic Kidney Disease (CKD)
- Acute Kidney Injury (AKI)
- Kidney Failure
- Kidney Stones

## Parameters Analyzed
- Age
- Blood Pressure
- Glucose
- Creatinine
- Blood Urea
- Sodium
- Potassium
- Hemoglobin
- Albumin

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib

## Workflow
1. Enter patient medical details.
2. Data is processed and analyzed.
3. Risk score is calculated.
4. Disease risk category is generated.
5. Health assessment report is displayed.
